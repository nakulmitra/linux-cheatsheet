# Cut, Copy & Paste in Vim

## Cut (Delete and store in buffer)
| Command | Description |
|---------|-------------|
| `dd` | Cut (delete) a line |
| `ndd` | Cut `n` lines |
| `dG` | Cut until end of file |

## Copy
| Command | Description |
|---------|-------------|
| `yy` | Copy a line |
| `nyy` | Copy `n` lines |
| `yG` | Copy until end of file |

## Paste
| Command | Description |
|---------|-------------|
| `p` | Paste **after** the cursor/line |
| `P` | Paste **before** the cursor/line |
