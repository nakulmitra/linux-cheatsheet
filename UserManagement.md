# User Management

| Command | Description |
|--------|-------------|
| `whoami` | Displays current user. |
| `id` | Shows user ID and group ID. |
| `adduser <user>` | Adds a new user. |
| `passwd <user>` | Changes user password. |
| `su <user>` | Switches to another user. |
| `sudo <command>` | Runs a command as root/superuser. |
